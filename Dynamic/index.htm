Welcome to the Anderson's Creek Primary School LOTE (Italian) web site. I have created this site to help children (prep to grade 6) practice their Italian pronunciation and vocabulary. The site was designed for the children to use in class on the intranet. The voice files are large and if used at home with a low speed modem will take some time to down load so please be patient. My students love the site and it was placed on the internet to allow them access from home. It will be best viewed using a 800 x 600 display. You are currently viewing the dynamic version.
